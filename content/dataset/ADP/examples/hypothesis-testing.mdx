# ADP 예제문제 2 - 가설검정

## 문제 설명

일표본 t-검정과 이표본 t-검정을 수행하시오.

## 데이터 준비

```python
import pandas as pd
import numpy as np
import scipy.stats as stats

np.random.seed(42)
group_a = np.random.normal(80, 10, 30)
group_b = np.random.normal(75, 12, 35)
```

## 문제 1: 일표본 t-검정

**Q1.** 그룹 A의 평균이 85와 다른지 검정하시오.

<CollapsibleCodeCell code={`print("귀무가설을 기각하지 않습니다.")`} />

## 문제 2: 이표본 t-검정

**Q2.** 두 그룹의 평균이 같은지 검정하시오.

<CollapsibleCodeCell language="python" defaultCollapsed={true}>
```python
t_stat, p_value = stats.ttest_ind(group_a, group_b)
print(f"t-통계량: {t_stat:.4f}")
print(f"p-값: {p_value:.4f}")

if p_value < 0.05:
    print("두 그룹의 평균이 다릅니다.")
else:
    print("두 그룹의 평균이 같다고 볼 수 있습니다.")
```
